# Amdox Task Manager - Full-Stack Application

**Amdox Task Manager** is a task management platform inspired by Jira, designed to help teams **organize, track, and manage tasks efficiently**. This project was built as a full-stack application with **React.js frontend**, **Spring Boot backend**, and **MySQL database**.

## Project Overview

Building this project has been a **journey from learning to implementation**. It helped me understand **how frontend, backend, security, and databases work together** in a production-like environment. Users can **register, login, create tasks, track workflows, and manage tasks** through a responsive React interface connected to Spring Boot REST APIs.

---

## Tech Stack

- **Frontend:** React.js, Axios, Bootstrap/ Tailwind CSS  
- **Backend:** Spring Boot, Spring Security, REST APIs  
- **Database:** MySQL  
- **API Testing:** Postman  
- **Build Tool:** Maven (Backend)  

---

## Key Learnings

- Implemented **Spring Security** for secure authentication  
- Designed and developed **RESTful APIs** with Spring Boot
